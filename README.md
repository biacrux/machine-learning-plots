# Machine Learning Plots

## Requisitos

Para correr este projeto, é necessário instalar o **Jupyter Notebook** e a extensão do Jupyter no **Visual Studio Code**.

## Instalação

### 1. Instalar o Jupyter
```bash
pip install jupyter
```

### 2. Visual Studio Code
- Abre o Visual Studio Code
- Vai a **Extensions** (Ctrl+Shift+X)
- Pesquisa por **Jupyter** e instala a extensão oficial da Microsoft

## Como ver os gráficos

Os gráficos não aparecem todos de uma vez — é necessário **correr cada célula individualmente**.

Para isso:
- Clica em cada célula e prime **Shift + Enter** para executar
- Ou usa o botão **▶ Run Cell** que aparece ao lado de cada célula

## Estrutura do projeto
- Os ficheiros `.ipynb` contêm o código e os gráficos
